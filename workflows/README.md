# Workflows

Each subdirectory is a self-contained analysis. Create a new folder here for each project you work on.

```
workflows/
  my_montana_drought/
    montana_drought.py          # your script
    output/                     # generated by the script (data gitignored, viz tracked)
      harmonized_*.tif
      harmonized_visualization.png
      harmonized_visualization.html
```

## Starting a new workflow

1. Create a folder: `workflows/my_project_name/`
2. Create a script inside it, following the pattern from `examples/colorado_fire_risk/`
3. Run from the repo root:

```bash
python workflows/my_project_name/my_script.py
```

Outputs will automatically land in `workflows/my_project_name/output/`.

## Template

```python
import sys
from pathlib import Path

# Walk up to find the repo root (dir containing src/geospatial_harmonizer.py).
# Depth-agnostic — works regardless of how nested this script is.
_repo_root = next(p for p in Path(__file__).resolve().parents
                  if (p / "src" / "geospatial_harmonizer.py").exists())
sys.path.insert(0, str(_repo_root))

from src.geospatial_harmonizer import DatasetSpec, ExampleWorkflow, run_harmonization_example

DATASETS = [
    DatasetSpec(
        name="my_raster",
        url="https://example.com/data.tif",
        data_type="raster",
    ),
    DatasetSpec(
        name="my_vector",
        url="https://example.com/boundaries.zip",
        data_type="vector",
        rasterize=True,
    ),
]

workflow = ExampleWorkflow(
    name="my_project",
    datasets=DATASETS,
    target_crs="EPSG:4326",
    target_extent=(-120.0, 35.0, -100.0, 50.0),  # your bounding box
    target_resolution=0.00243,                     # ~270 m
    output_dir=Path(__file__).parent / "output",   # always relative to this script
    create_visualization=True,
    verbose=True,
)

output_files, interactive_map = run_harmonization_example(workflow)
```
